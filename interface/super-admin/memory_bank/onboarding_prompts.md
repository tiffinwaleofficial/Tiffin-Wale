**10. Scope Onboarding Prompts:**

*   **Frontend Scope Onboarding Prompt**:
    "I'm starting work on the Frontend scope. Please provide a summary of the UI architecture, key components, state management patterns, and how data is fetched and displayed. Include relevant file paths for pages, components, contexts, and hooks."
*   **Backend Scope Onboarding Prompt**:
    "I'm starting work on the Backend scope. Please provide an overview of the Firebase integration, authentication flow, data models (Firestore), and any server-side logic or API routes. Include relevant file paths for Firebase config, middleware, and AI services."
*   **DevOps Scope Onboarding Prompt**:
    "I'm starting work on the DevOps scope. Please provide details on the deployment process to Google App Engine, environment variable management, and any CI/CD configurations. Include relevant file paths for `app.yaml`, `dispatch.yaml`, and `package.json` scripts."
*   **Database Scope Onboarding Prompt**:
    "I'm starting work on the Database scope. Please provide information on the Firestore data structure, security rules, indexing, and how data is accessed and manipulated from the application. Include relevant file paths for Firebase config, hooks, and rules."
*   **Integrations Scope Onboarding Prompt**:
    "I'm starting work on the Integrations scope. Please provide details on how Firebase (Auth, Firestore) and Genkit AI are integrated into the application, including configuration and usage patterns. Include relevant file paths for integration-related files."
